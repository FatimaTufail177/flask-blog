Flask Blog Web Application

A full-stack blog web application built with "Flask" that allows users to register, log in, and create, edit, and delete blog posts. The system includes authentication, role-based authorization, and database integration using SQLAlchemy.

This project demonstrates real-world backend and full-stack web development using Python.

---

Features

* User registration and login
* Secure password handling
* Create, read, update, and delete (CRUD) blog posts
* Role-based access (only authors can edit or delete their posts)
* Relational database using SQLAlchemy ORM
* Dynamic frontend using Jinja templates
* Clean and responsive UI

---

Tech Stack

* **Backend:** Python, Flask
* **Database:** SQLite, SQLAlchemy
* **Frontend:** HTML, CSS, Bootstrap, Jinja2
* **Authentication:** Flask sessions, password hashing
* **APIs:** REST-style backend routes

How to Run Locally

1. Clone the repository

```
git clone https://github.com/FatimaTufail177/flask-blog.git
```

2. Go into the project folder

```
cd flask-blog
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the application

```
python app.py
```

5. Open in browser

```
http://127.0.0.1:5000
```
